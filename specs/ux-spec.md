ASCII Layout (with LED + Encoder separated)
+---------------------------------------------------+
|                 Braids (macro osc)                |
|---------------------------------------------------|
|  (A) [ LED DISPLAY ]       (B) EDIT Encoder       |
|  (e.g. "VOWL")         (rotary+push)              |
|                                                   |
|       (C)FINE        (D)COARSE        (E)FM       |
|        knob            knob            knob       |
|                                                   |
| (F)TIMBRE         (G)MODULATION        (H)COLOR   |
|   knob               knob                knob     |
|                                                   |
|                                                   |
+---------------------------------------------------+
|  A: LED Display (mode / model / menu)             |
|  B: EDIT Encoder (rotate + push to select)        |
|  C: Fine frequency knob                           |
|  D: Coarse frequency knob                         |
|  E: FM attenuverter                               |
|  F: Timbre knob                                   |
|  G: Modulation attenuverter                       |
|  H: Color knob                                    |
|                                                   |
+---------------------------------------------------+



Alternate Graphical ASCII “knob sketch” 
+---------------------------------------------------+
|                 Braids (macro osc)                |
|---------------------------------------------------|
|                                                   |
| [ VOWL ]                        (@) EDIT          |
|   LED display                   encoder           |
|                                                   |
|     ( ) FINE        ( ) COARSE        ( ) FM      |
|                                                   |
|     ( ) TIMBRE      ( ) MODULATION    ( ) COLOR   |
|                                                   |
+---------------------------------------------------+

Legend:
	•	[@] = Rotary encoder (clickable)
	•	[ VOWL ] = LED alphanumeric display (mode/params)
	•	( ) = Knobs (fine, coarse, FM, timbre, modulation, color)
